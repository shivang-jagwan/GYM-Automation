# Frontend Setup (React)

## Requirements
- Node.js 20 LTS

## Setup Steps
1. Open the `frontend` folder in terminal.
2. Run:
   ```
   npx create-react-app .
   npm install axios
   ```
3. Start the frontend:
   ```
   npm start
   ```

## Notes
- The frontend uses Axios to call backend APIs.
- All UI is functional, simple, and admin-only.
- No Vite, no Next.js, no heavy UI libraries.
